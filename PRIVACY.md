# 拾图隐私政策

生效日期：2026 年 7 月 18 日

拾图是一款本地运行的 Windows 截图工具。本政策说明应用处理截图、OCR 结果和配置数据的方式。

## 我们处理的数据

- 当你主动发起截图、钉图或 OCR 时，应用会读取你选择的屏幕区域及其中可能包含的文字和图像。
- 应用会在本机保存你选择保存的截图文件，以及运行所需的本地设置，例如截图格式、快捷键和 OCR 偏好。
- 应用不会要求你创建账户，也不会收集姓名、邮箱、电话号码、精确位置、联系人、支付信息或广告标识符。

## 数据如何使用

- 截图内容仅用于完成你请求的截图、标注、复制、保存、钉图和 OCR 功能。
- OCR 在本机通过 Windows 提供的能力处理；拾图不会将截图内容或 OCR 结果上传至开发者服务器或第三方分析服务。
- 应用不包含广告、行为分析或遥测上传功能。

## 数据存储与共享

- 你主动保存的截图由你选择的本地位置管理。
- 应用设置保存在你的 Windows 本地应用数据目录中。
- 除非你主动使用 Windows 共享、复制、保存或向 GitHub 问题反馈提交内容，拾图不会向开发者或其他第三方发送截图、OCR 结果或设置数据。

## 你的选择

- 你可以随时取消截图或 OCR 操作。
- 你可以在应用设置中调整功能选项，并可自行删除已保存的截图和本地设置。
- 通过 GitHub 提交问题反馈时，请不要在问题内容、截图或附件中包含不希望公开的信息；该反馈受 GitHub 的隐私政策和服务条款约束。

## 政策更新与联系

本政策可能会随功能变化更新，更新后的版本会发布在本页面。若有隐私相关问题，请通过 [GitHub Issues](https://github.com/dripai/shitu/issues) 联系我们。
